# Inadimplência B2B no Brasil — Relatório Público

> Dados consolidados de fontes públicas sobre inadimplência no crédito B2B brasileiro. Atualizado mensalmente. Acesso gratuito.

**[relatorio.besinky.com](https://relatorio.besinky.com)**

---

## O projeto

22,7% das empresas B2B brasileiras operam com dívidas em atraso. R$ 150,4 bilhões em aberto. 94 dias de prazo médio antes da negativação.

Esses dados existem — espalhados em relatórios do BACEN, Serasa, SPC Brasil e Boa Vista. O que não existia era um lugar que os consolidasse, visualizasse e tornasse acessíveis para qualquer pessoa: jornalistas, analistas de crédito, CFOs, pesquisadores.

Este projeto foi construído para ser essa referência.

---

## O que tem dentro

- **Evolução histórica desde 2020** — série temporal mensal com o impacto do COVID, recuperação e alta recente
- **Breakdown por setor** — Construção Civil, Varejo, Agronegócio, Indústria, Transporte e mais
- **Breakdown por porte** — de MEI a grandes empresas, com variação anual
- **Filtros interativos** — setor, porte e período propagam para todos os gráficos simultaneamente
- **As 5 causas mais comuns de inadimplência B2B** — com ranking e dados de impacto
- **O que os melhores fazem diferente** — práticas de operações com inadimplência abaixo da média
- **Calculadora de custo** — estimativa de impacto baseada no tamanho da carteira
- **Gate de acesso** — captura de leads qualificados antes do acesso ao relatório completo

---

## Stack

**HTML · CSS · JavaScript · Chart.js · Vercel**

Projeto intencional mente sem framework. Leve, rápido, sem build step. Os dados vivem em um único arquivo JSON atualizado mensalmente por um agente automatizado.

---

## Contexto

Este projeto é parte da estratégia de **inbound e autoridade de dados** da [Sinky](https://besinky.com) — plataforma de infraestrutura de decisão para empresas que operam crédito.

A Sinky não tem dados proprietários de inadimplência. O posicionamento como referência de mercado vem da curadoria e apresentação de dados públicos que o setor já produz — mas ninguém organiza.
